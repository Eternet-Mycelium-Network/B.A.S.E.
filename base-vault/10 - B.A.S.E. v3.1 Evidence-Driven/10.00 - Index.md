# B.A.S.E. v3.1 — Evidence-Driven Architecture

> *Trocar ML + heurística por Evidence + Contracts + Solver formal.*

## Princípio

| Antes (v2) | Depois (v3.1) |
|-----------|---------------|
| "Isto é uma GPU (72%)" | "Observei MMIO write em 0x10000000, IRQ linha 5, DMA" |
| ML + heurística | Contratos formais |
| Confidence na evidência | Evidence = fato puro, Assessment = separado |
| PCB final | Reference Design (obrigatório) + Engineering Draft (opcional) |

## Arquitetura

```text
Firmware / Trace / HIL
       ↓
[Evidence DB] — fatos puros
       ↓
[Evidence Assessment] — opinião (opcional)
       ↓
[BIR] — Behavioral IR
       ↓
[Contract Engine] — requires: dma, irq, mmio, latency < 500ns
       ↓
[Contract Solver] — matching formal (requires × provides)
       ↓
[Reference Design] — cpu: rp2350, memory: sram (SAÍDA PRINCIPAL)
       ↓
[KiCad Generator — OPCIONAL] → Engineering Draft
       ↓
[Validation] — contratos satisfeitos?
```
