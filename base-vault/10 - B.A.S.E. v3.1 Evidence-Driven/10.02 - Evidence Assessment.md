# Evidence Assessment — Opinião Separada

> *Confiança e justificativas, separadas dos fatos.*

## Formato

```yaml
# assessment.yaml — OPCIONAL, separado da evidência
assessment:
  - evidence_id: ev_001
    sources:
      - disassembly (capstone)
      - trace (saleae)
    confidence: 0.95
    justification:
      - confirmed_by_disasm
      - observed_3_times
      - matches_dtb_entry

  - evidence_id: ev_002
    sources:
      - disassembly
    confidence: 0.70
    justification:
      - confirmed_by_disasm
      - single_observation
```

## Por que separar?

| Camada | Mutável? | Contém |
|--------|----------|--------|
| Evidence DB | ❌ Imutável | Fatos observados |
| Assessment | ✅ Evolui | Confidence, fontes, justificativas |

A evidência nunca muda. A interpretação pode ser refinada.

## Fontes de Evidência

| Fonte | Confiança típica | Descrição |
|-------|-----------------|-----------|
| Disassembly (Capstone) | 0.95 | Instrução ARM64 confirmada |
| Trace Saleae | 0.90 | Captura direta do barramento |
| Driver Linux | 0.70 | Código fonte do driver |
| Datasheet | 0.60 | Documentação oficial |
| DTB | 0.50 | Device Tree |
| HIL Probe | 0.95 | Captura hardware-in-the-loop |
| UART log | 0.60 | Log de debug |
