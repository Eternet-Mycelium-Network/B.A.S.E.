# Passo 11 — Trace Replay Engine

> *Replay de trace contra BIR. Gera contract_violations.json.*

## Problema

Hoje a validação compara **traces brutos** (original vs novo). Não verifica contratos contra execução real.

## Solução

```bash
base replay trace.csv --contracts contracts.yaml
# → contract_violations.json
```

## Arquitetura

```text
trace.csv
   ↓
[Trace Replay Engine]
   ↓
contracts.yaml → verifica sequências → latências → ordenação
   ↓
contract_violations.json
```

## Formato de Saída

```json
{
  "replay": {
    "sequences_checked": 47,
    "passed": 44,
    "violations": [
      {
        "contract": "dma_transfer",
        "step": 3,
        "expected": "IRQ within 500ns of DMA_COMPLETE",
        "actual": "IRQ after 1.2µs",
        "severity": "warning"
      }
    ]
  }
}
```

## Implementação

```rust
struct TraceReplay {
    contracts: Vec<SequenceContract>,
    trace: DeviceTrace,
}

impl TraceReplay {
    fn run(&self) -> ReplayReport {
        let mut violations = Vec::new();

        for contract in &self.contracts {
            let sequences = self.find_patterns(&contract.steps);

            for seq in &sequences {
                // Verificar ordem
                if contract.order == OrderConstraint::Strict {
                    if !self.is_ordered(seq) {
                        violations.push(Violation {
                            contract: contract.name.clone(),
                            kind: ViolationKind::WrongOrder,
                            detail: "Events out of order".into(),
                        });
                    }
                }

                // Verificar latência total
                let total_ns = seq.last().timestamp_ns - seq.first().timestamp_ns;
                if total_ns > contract.max_total_ns {
                    violations.push(Violation {
                        contract: contract.name.clone(),
                        kind: ViolationKind::TotalLatency,
                        detail: format!("{}ns > {}ns", total_ns, contract.max_total_ns),
                    });
                }

                // Verificar latência entre passos
                for pair in seq.windows(2) {
                    let dt = pair[1].timestamp_ns - pair[0].timestamp_ns;
                    if dt > contract.max_step_ns {
                        violations.push(Violation {
                            contract: contract.name.clone(),
                            kind: ViolationKind::StepLatency,
                            detail: format!("step latency {}ns > {}ns", dt, contract.max_step_ns),
                        });
                    }
                }
            }
        }

        ReplayReport {
            sequences_found: sequences_count,
            passed: passed_count,
            violations,
        }
    }

    /// Encontra ocorrências de um padrão de eventos no trace
    fn find_patterns(&self, steps: &[EventStep]) -> Vec<Vec<&TraceEvent>> {
        let mut occurrences = Vec::new();
        let mut i = 0;
        while i < self.trace.events.len() {
            if self.matches(&self.trace.events[i], &steps[0]) {
                let mut seq = vec![&self.trace.events[i]];
                let mut j = 1;
                while j < steps.len() && i + j < self.trace.events.len() {
                    if self.matches(&self.trace.events[i + j], &steps[j]) {
                        seq.push(&self.trace.events[i + j]);
                    } else { break; }
                    j += 1;
                }
                if seq.len() == steps.len() {
                    occurrences.push(seq);
                    i += steps.len();
                    continue;
                }
            }
            i += 1;
        }
        occurrences
    }
}
```

```bash
base replay trace.csv --contracts contracts.yaml
base replay trace.csv --contracts contracts.yaml --output violations.json
base replay trace.csv --bir device.bir.yaml  # extrai contratos do BIR
```
