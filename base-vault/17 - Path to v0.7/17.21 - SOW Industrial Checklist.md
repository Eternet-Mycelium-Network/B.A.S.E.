# SOW checklist — industrial (v0.7)

> Copiar para o contrato. **Não** SaaS turnkey. Humano no loop.
> Base: [[16 - Path to v0.6/16.21 - SOW Industrial Checklist|SOW v0.6]].

## Escopo (checklist)

- [ ] Analyze + Evidence (RP e/ou STM32)
- [ ] Capstone/disasm no wedge STM32 se acordado (blob sintético ≠ Thumb silício)
- [ ] Dual peripheral STM32 (USART + SPI2) se acordado — SPI1 page-clash documentado
- [ ] Goldens event-graph / prove no path STM32 (smoke `diff`)
- [ ] Design com preferência de fabricante
- [ ] Draft PCB pin-aware (`NOT FABRICABLE`)
- [ ] Reconstruct: report `stop_reason` / estagnação (≠ auto-fix)
- [ ] Demo: `run.sh` / `pilot_stm32` / `run_w1_spi.sh` / `base hil` EXPERIMENTAL
- [ ] Relatório de viabilidade + handoff

## Não-escopo

- [ ] PCB fabricável / Gerber de fábrica
- [ ] Flash HIL “production” / silício sem aprovação escrita
- [ ] `libusb` obrigatório no CI do Cliente
- [ ] SaaS turnkey / 24×7
- [ ] Substituir gate RP pelo STM32
- [ ] Auto-fix completa via `reconstruct --continuous`
- [ ] Amiga/CD32 como entregável de release

## HIL (add-on)

| Item | Default |
|------|---------|
| CLI | `base hil enumerate\|flash` — EXPERIMENTAL |
| Dry-run | `--mock-flash` → `mock_dry_run` |
| Real | feature `hil_programmer` + `ALLOW_FLASH` + CMD do operador |

## Aceite

1. Gates RP verdes no Prestador (`run.sh` + `run_t1_b2.sh`).  
2. Se STM32 no escopo: smoke `pilot_stm32/run.sh` (goldens) e, se dual SPI, `run_w1_spi.sh` — ou bloqueio honesto.  
3. Relatório sem claims da seção Não-escopo; reconstruct sem claim de auto-fix.
