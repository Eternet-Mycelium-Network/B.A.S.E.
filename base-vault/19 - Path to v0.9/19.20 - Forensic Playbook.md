# Playbook forense — 1 página (v0.9)

**Objetivo:** Evidence → contratos → Design (RP + STM32 USART/SPI2/I2C1) + pins I2C + goldens `diff` + smoke triplo + reconstruct honesto + HIL CLI EXPERIMENTAL; engenheiro no loop.

## Setup (12–15 min)

```bash
git clone https://github.com/bmcc-DEV/B.A.S.E..git
cd B.A.S.E.
cargo build -p base-cli

# Gates permanentes (RP goldens = diff, sem overwrite)
./examples/pilot/run.sh
./examples/pilot/run_t1_b2.sh

# STM32 USART (Capstone + pins + goldens)
./examples/pilot_stm32/run.sh

# STM32 dual USART+SPI2 + pins PB13/14/15
./examples/pilot_stm32/run_w1_spi.sh

# STM32 dual USART+I2C1 + pins PB6/PB7 + goldens expected_i2c/
./examples/pilot_stm32/run_x3_i2c.sh

# STM32 triple USART+SPI2+I2C1 (Y3)
./examples/pilot_stm32/run_y3_triple.sh

# Reconstruct (estagnação honesta — ≠ auto-fix)
base reconstruct examples/pilot_stm32/out/analyze/hardware_spec.yaml \
  --continuous --threshold 0.99 -o /tmp/recon_demo

# HIL CLI (host-only)
base hil enumerate -o /tmp/hil_demo
base hil flash /dev/null --mock-flash -o /tmp/hil_demo
```

## Mostrar ao cliente

1. **RP** — UART + SPI; **goldens** `diff` vs `expected/`.
2. **STM32 USART** — Capstone; PA9/PA10; goldens STM32.
3. **STM32 SPI2** — `@ 0x40003800` + labels PB13/14/15 `NOT FABRICABLE`.
4. **STM32 I2C1** — `@ 0x40005400` + PB6/PB7 + goldens `expected_i2c/` (`diff`).
5. **Triple** — classify 3 páginas (`run_y3_triple.sh`); design ST → `STM32F103C8`.
6. **Reconstruct** — `stop_reason` / estagnação; **não** auto-fix.
7. **HIL CLI** — Simulated / `--mock-flash`; nunca production.
