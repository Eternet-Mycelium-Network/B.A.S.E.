# SOW checklist — industrial (v1.2)

> Copiar para o contrato. Humano no loop.  
> Base: [[21 - Path to v1.1/21.21 - SOW Industrial Checklist|SOW v1.1]] + [[22.30 - SOW Industrial Gate]].

## Escopo forense (sempre)

- [ ] Analyze + Evidence (RP e/ou STM32)
- [ ] Specter `base study` / reconstruct com `stop_reason` se acordado
- [ ] Goldens `diff` / evolve métricas / hil **host** / pcb **draft**
- [ ] Relatório de viabilidade + handoff

## Escopo industrial (só se Gate verde)

Marcar **apenas** o que passou [[22.30 - SOW Industrial Gate]]:

- [ ] **HIL lab** — pré-condições A1–A5  
- [ ] **PCB engenharia** — pré-condições B1–B5  
- [ ] **Auto-fix parcial** — pré-condições C1–C5  

## Não-escopo (default)

- [ ] PCB fabricável sem EE / Gerber automático do draft
- [ ] HIL production / flash no CI / SaaS turnkey
- [ ] Auto-fix completa / “conserta sozinho”
- [ ] ASIC drop-in / Amiga-CD32 como entregável
- [ ] Implementação PaleoCLI

## Aceite

1. Gates: `run.sh` + `run_t1_b2.sh` (+ `run_study.sh` se study no escopo).  
2. Claims industriais: só os marcados acima **e** com anexos do Gate.  
3. Relatório sem claims Não-escopo.
