# Target ISA Matrix

| TargetIsa | Alias CLI | Endian | Reg class | Sprint emit | Consola / uso tipico |
|-----------|-----------|--------|-----------|-------------|----------------------|
| `X86_64` | `x86_64`, **`amd64`** | LE | GP 64 | R2 | Host moderno |
| `Arm` | `arm` | LE | GP 32 | R3 | Embedded / PS Vita class* |
| `AArch64` | `arm64`, `aarch64` | LE | GP 64 | R3 | Phones / Apple Silicon |
| `Mips` | `mips` | BE/LE | GP 32 | R4 | N64 / PS1 class* |
| `Ppc` | `ppc`, `powerpc` | BE | GP 32 | R4 | GameCube / iMac G3 class* |
| `Sparc` | `sparc` | BE | GP 32 | R4 | Workstations / SPARCstation |
| `SuperH` | `sh`, `superh`, `sh2`, `sh4` | LE | GP 16/32 | R5 | **Saturn (SH-2)** · Dreamcast (SH-4) |

\* “class” = envelope de ISA, **não** runtime de consola completo.

## Frontend

| Source | Sprint |
|--------|--------|
| x86-32 raw / ELF `.text` | R1 |
| PE COFF + relocs | future |
| Win32 imports / SEH | **fora** v1.7 (`win32_abi_complete: false`) |

## Política amd64

`amd64` e `x86_64` resolvem para o **mesmo** `TargetIsa::X86_64`. Documentação e CLI aceitam ambos; testes usam um único golden.
