## Demo

```bash
cargo test -p base-recomp --test r4_classic_risc
base recomp lift --hex B8010000000502000000C3 --name add3 --target mips -o output/r4_mips
base recomp lift --hex 90c3 --target ppc -o output/r4_ppc
base recomp lift --hex 90c3 --target sparc -o output/r4_sparc
```
