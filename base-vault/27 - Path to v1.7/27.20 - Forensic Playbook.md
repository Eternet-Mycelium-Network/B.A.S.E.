# Forensic Playbook — Static Recomp (v1.7)

## Demo R5 (SuperH SH-2 / SH-4)

```bash
cargo test -p base-recomp --test r5_superh
base recomp lift --hex B8010000000502000000C3 --target sh2 -o output/r5
base recomp lift --hex 90c3 --target sh4 -o output/r5_sh4
```

Path v1.7 **fechado** em código/vault. Tag `v1.7.0-rc` = passo de release.

## Demo R4 (MIPS / PPC / SPARC goldens)

```bash
cargo test -p base-recomp --test r4_classic_risc
base recomp lift --hex B8010000000502000000C3 --target mips -o output/r4
base recomp lift --hex 90c3 --target powerpc -o output/r4_ppc
base recomp lift --hex 90c3 --target sparc -o output/r4_sparc
```

Assemble live clássico: golden-only até binutils cross no PATH.

## Demo R3 (ARM goldens + assemble-only)

```bash
cargo test -p base-recomp --test r3_arm_family
base recomp assemble-arm --hex B8010000000502000000C3 --name add3 -o output/r3
base recomp lift --hex 90c3 --target arm64 -o output/r3_a64
```

AArch64: golden emit estável; assemble live requer `aarch64-*-as` (ainda não no lab default).

## Demo R2 (host x86_64 + GNU as/cc)

```bash
cargo test -p base-recomp --test r2_x86_64
base recomp roundtrip --hex B8010000000502000000C3 --name add3 --expect 3 -o output/r2
# mov eax,1 ; add eax,2 ; ret → eax==3
```

Nota: GAS usa `#` para comentários; banners com `;` são stripped antes do `as`.
